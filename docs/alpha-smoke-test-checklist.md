# Private Alpha Smoke Test Checklist

Use this quick checklist during private alpha testing. Note anything confusing, broken, or unexpectedly slow.

- [ ] Sign up for a new account or sign in to an existing account.
- [ ] Create your first profile.
- [ ] Create a vehicle.
- [ ] Use the single show calculator.
- [ ] Save the run.
- [ ] Redeem code `TESTER101`.
- [ ] Confirm pro/tester features are unlocked.
- [ ] Open the dashboard.
- [ ] Create or edit a venue.
- [ ] View venue history.
- [ ] Create a tour.
- [ ] Confirm there are no obvious broken routes or 500 errors.
