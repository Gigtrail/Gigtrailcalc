# Private Alpha Smoke Test Checklist

Use this quick checklist during private alpha testing. Note anything confusing, broken, or unexpectedly slow.

- [ ] Sign up for a new account or sign in to an existing account.
- [ ] Complete onboarding without a dead end.
- [ ] Create one usable act profile with home base and calculator defaults.
- [ ] Create or confirm one default vehicle that the calculator can use.
- [ ] Calculate one single show.
- [ ] Save the single-show calculation.
- [ ] Reopen the saved show and confirm the saved financial snapshot matches the original result.
- [ ] Save or use one past-dated completed show with known totals.
- [ ] Open the dashboard and confirm totals match past saved shows only.
- [ ] Confirm drafts, future shows, tours, and projections do not affect dashboard totals.
- [ ] Redeem code `TESTER101`.
- [ ] Confirm tester/pro access is unlocked after redemption.
- [ ] Sign in as `thegigtrail@gmail.com` and confirm admin access still works.
- [ ] Submit feedback while signed in.
