1. Resolve venue identity using:

   * venueName
   * city
   * country

2. Generate:
   normalizedVenueKey =
   lower(trim(venueName)) + "|" + lower(trim(city)) + "|" + lower(trim(country))

3. Check for existing venue:
   WHERE userId AND normalizedVenueKey match

4. If NOT found:

   * Create venue
   * Set:
     venueType = "personal"
     userId
     name
     city
     country
     normalizedVenueKey

5. If found:

   * Use existing venue.id

6. Save run with:
   venueId (REQUIRED — never null)
