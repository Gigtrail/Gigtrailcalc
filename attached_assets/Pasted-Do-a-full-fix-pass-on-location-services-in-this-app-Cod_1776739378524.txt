PART 5 — FIX FUEL CALCULATION USING REAL DISTANCE
==================================================

Fuel depends on route distance, so fix the full dependency chain.

Use one helper only, for example:
resolveFuelCost({
  distanceKm,
  fuelEfficiencyLPer100Km,
  fuelPricePerLitre
})

Formula:
litresUsed = (distanceKm / 100) * fuelEfficiencyLPer100Km
fuelCost = litresUsed * fuelPricePerLitre

Rules:
- do not calculate fuel if distance is missing
- do not silently show $0 unless truly zero by design
- if route is unavailable, show a clear state instead
- ensure expenses total and result page use the same computed fuel value
