Update the Tour Calc results UI so it clearly shows **how much each band member is earning from the gig fees across the tour**, both **per show** and **as a total for the full tour**.

## Goal

I want to be able to quickly see:

* each member in the selected act profile
* what their agreed fee is
* whether that fee is **per show** or **per tour**
* how much they are expected to earn based on the currently entered shows
* a total payout summary for all members combined
* remaining profit after member payouts

This needs to work inside the existing **Tour Calc** flow and appear in the results/dashboard area.

---

## Core logic

Use the selected **Act Profile** as the source of truth.

Each member should already have or now support:

* `name`
* `role`
* `fee_amount`
* `fee_type`

Where `fee_type` can be:

* `per_show`
* `per_tour`
* `none`

### Earnings calculation rules

For the current tour:

* Count only shows that are marked as actual booked/played income shows
* Ignore empty days with “No show booked”
* For each member:

  * if `fee_type === "per_show"`
    `member_total = fee_amount * qualifying_show_count`
  * if `fee_type === "per_tour"`
    `member_total = fee_amount`
  * if `fee_type === "none"`
    `member_total = 0`

Also calculate:

* `total_member_payout = sum of all member totals`
* `tour_profit_after_member_fees = net_profit_before_member_fees - total_member_payout`

If there is already a final income/profit number in the calculator, subtract member payouts from that final figure rather than creating conflicting totals.

---

## What I want shown in the UI

Add a new results card called:

### `Member Earnings`

Inside that card, show one row per member.

Each row should include:

* Member name
* Role
* Fee setup label
  Examples:

  * `$250 / show`
  * `$800 / tour`
  * `No fee set`
* Number of qualifying shows counted
* Estimated earnings for this tour

### Example row format

* Julian James — Guitar/Vocals — $300/show — 8 shows — **$2,400**
* Beci Kate — Percussion/Vocals — $250/show — 8 shows — **$2,000**
* Bass Player — Bass — $800/tour — 8 shows — **$800**

At the bottom of the card show:

* `Total Member Payout`
* `Profit After Member Fees`

## Qualifying show rules

A show should count toward per-show earnings only if it is a real income show.

Use existing tour show data and count only shows that:

* have a booked/past/confirmed status
* and/or have a fee/income value greater than 0
* are not blank placeholders

Do not count travel days or empty tour days.
