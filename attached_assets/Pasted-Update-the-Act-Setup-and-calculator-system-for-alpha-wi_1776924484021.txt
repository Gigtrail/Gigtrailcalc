Add a new band-level field:

payoutMode: "fixed" | "split"
Default: "fixed"

---

### UI

Add a section in Act Setup:

“How are you paying the band?”

Options:
(•) Fixed Pay
( ) Profit Split

---

### Behavior

If payoutMode = "fixed":

* Show existing member fee inputs (current system)

If payoutMode = "split":

* Hide ALL member fee inputs

  “Remaining profit will be split evenly between X members”

## 3. Add Minimum Act Take-Home

Add a new band-level numeric field:

minimumActTakeHome (number, optional, default = 0)

Label:
Minimum Act Take-Home

Helper text:
“The amount the act keeps before any remaining profit is considered surplus or split”

---

### Placement

* This should sit at band level (not per member)
* Visible in BOTH payout modes

grossIncome

## 5. Results UI (Plain English — very important)

---

### Fixed Mode Display

* “You need $X to cover expenses and band fees”
* “You need $Y to also hit your minimum take-home”
* “Surplus after target: $Z”

Where:
X = expenses + memberFees
Y = expenses + memberFees + minimumActTakeHome
Z = remainingSurplus

---

### Split Mode Display

* “Minimum take-home target: $X”
* “Split pool: $Y”
* “Per member: $Z”

Where:
X = minimumActTakeHome
Y = splitPool
Z = perMember
