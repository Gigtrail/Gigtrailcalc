## 3. Automatic route calculation

When user clicks “Calculate Gig”:

### Required inputs
- Origin location
- Destination location
- Vehicle fuel consumption
- Fuel price
- Return trip toggle

### Route calculation
Use Google routing to calculate:
- one-way distance in km
- one-way driving duration
- if return trip is enabled:
  - total distance = one-way distance × 2
  - total duration = one-way duration × 2

### Output fields
Display:
- Distance (km)
- Estimated drive time
- Fuel used (L)
- Fuel cost ($)

If no route can be found:
- show friendly error:
  "We couldn't calculate the route for this run. Please check the locations."

---

## 4. Fuel calculation logic

Use:
fuel_used_litres = (total_distance_km * vehicle_l_per_100km) / 100
fuel_cost = fuel_used_litres * fuel_price

### Example
If:
- total distance = 400 km
- vehicle = 10 L/100km
- fuel price = 2.00

Then:
- fuel used = 40 L
- fuel cost = 80.00

Please ensure:
- all values are numeric
- currency formatting is correct
- no string concatenation bugs
